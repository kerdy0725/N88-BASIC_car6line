# N88-BASIC_car6line
N88BASICで作った6行のカーレースゲーム

![スクリーンショット 2024-06-07 231602](https://github.com/kerdy0725/N88-BASIC_car6line/assets/109938057/5a320f39-1c9b-4228-b076-bd17b8706a31)

## 動作環境
 * N88互換BASIC for Windows95 1.10
 * Windwos11で動作確認しました。
## 操作
 * a / s …　車(￥)の左右移動
 * w … スピードアップ
 * x …　スピードダウン
